# Customer Segmentation Dashboard

## Project Overview
This project analyzes customer purchasing behavior using Excel, MySQL, and Tableau. The goal of the project is to identify customer spending patterns, customer segments, preferred payment methods, and subscription behavior through interactive visualizations and business analysis.

---

## Tools Used
- Excel
- MySQL
- Tableau

---

## Data Preprocessing
Performed data cleaning and preprocessing using Excel:
- Handled missing values
- Removed duplicate records
- Corrected invalid values
- Standardized data formatting
- Created Customer Segment column

---

## SQL Analysis Performed
- Total Revenue Analysis
- Gender-wise Spending Analysis
- Category-wise Sales Analysis
- Customer Segment Analysis
- Payment Method Analysis
- Subscription Status Analysis
- Review Rating Analysis

---

## Dashboard Features
- Total Customers KPI
- Total Revenue KPI
- Average Spending KPI
- Gender Spending Analysis
- Customer Segment Analysis
- Category Analysis
- Payment Method Analysis
- Subscription Analysis
- Review Rating Analysis
- Interactive Dashboard Filters

---

## Key Business Insights
- High-value customers generated the highest revenue
- Certain product categories contributed more sales
- Subscribed customers showed better spending behavior
- Payment preferences varied among customers
- Customer ratings differed across categories
